# Jane's Fashion Blog

A clone of the W3Schools Fashion Blog template, enhanced with JavaScript DOM manipulation features as part of a web development assignment.


### Real-Time Search Filtering
A search bar at the top filters blog posts as the user types.

- Uses `querySelector` to grab the input element.
- Uses `innerText.includes()` to check if each post's text matches the query.
- Posts that don't match are hidden in real time.
